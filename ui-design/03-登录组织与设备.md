# 登录、组织与设备页面设计

覆盖页面：

- `/login`：Login.vue
- `/org/:slug`：Organization.vue
- `/org/:slug/devices`：DeviceManagement.vue

## 登录页

### 目标

让用户快速进入已有组织，或完成注册并创建/选择组织。页面只承载认证、注册、组织选择和语言切换，不展示营销内容。

### 布局

桌面端使用居中登录面板：

- 页面背景：`#F5F7FA`。
- 面板宽度：420px。
- 面板 padding：32px。
- 面板圆角：4px，边框 `#DCDFE6`。
- 产品名位于面板上方，字号 32px，字重 700。

移动端：

- 面板宽度 `calc(100vw - 32px)`。
- padding 24px。
- 顶部 logo 与语言切换保留。

### 登录表单

字段：

| 字段 | 控件 | 规格 |
|------|------|------|
| 账号 | 输入框 | 支持用户名或邮箱，32px 高。 |
| 密码 | 密码输入框 | 支持显示/隐藏。 |
| 语言 | 下拉或分段控件 | 中文 / English。 |

动作：

- 主按钮：登录。
- 次操作：注册账号。
- 成功后如用户只有一个组织，直接进入项目仪表盘；多个组织进入组织选择。

状态：

- 登录中：按钮 loading，表单不可重复提交。
- 失败：表单顶部错误条，显示后端错误信息。
- token 过期：跳转登录页并保留来源路径，登录后返回。

### 注册表单

字段：

- 用户名。
- 邮箱。
- 密码。
- 默认语言。

注册成功后进入组织创建/选择，不自动创建项目。

## 组织选择

### 触发

登录后用户属于多个组织时显示；用户无组织时引导创建组织。

### 布局

同登录面板，面板宽 520px。

组织列表项：

- 高度 64px。
- 左侧组织头像或首字母。
- 中间组织名称、slug、成员角色。
- 右侧进入箭头。

空态：

- 标题：暂无组织。
- 主操作：创建组织。

## 组织管理页

### 页面目标

管理组织基本信息、成员和组织下项目入口。设备管理独立页面，但在组织页保留入口。

### 页面结构

```
标题区：组织名称 / slug / 当前用户角色 / 编辑组织
Tabs：成员 / 项目 / 设置
```

### 成员 Tab

表格列：

| 列 | 宽度 | 说明 |
|----|------|------|
| 成员 | 240px | 头像、昵称、邮箱。 |
| 角色 | 120px | owner / admin / member。 |
| 状态 | 120px | active / invited。 |
| 加入时间 | 160px | 日期。 |
| 操作 | 160px | 修改角色、移除。 |

操作：

- 邀请成员：弹窗，输入邮箱和角色。
- 修改角色：下拉选择，保存后表格行内更新。
- 移除成员：危险确认弹窗。

权限：

- 非管理员只读成员列表。
- 不能移除自己作为唯一 owner 的组织。

### 项目 Tab

展示组织内项目列表：

- 项目名、slug、Git provider、默认分支、Issue 数、最近更新时间。
- 主操作：新建项目，跳转 `/projects/new`。

### 设置 Tab

字段：

- 组织名称。
- slug，只允许在创建后按后端规则更新。
- 组织级设置 JSON 的可视化摘要，不暴露未定义配置项。

保存动作：

- 使用 `PUT /api/v1/orgs/:slug`。
- 保存成功显示 toast，不离开页面。

## 设备管理页

### 页面目标

展示当前用户已登录设备，支持远程吊销和保留当前设备。

### 布局

标题栏：

- 标题：已登录设备。
- 副标题：展示当前账号的活跃设备。
- 右侧危险次按钮：注销其他设备。

设备卡片或表格：

| 列 | 说明 |
|----|------|
| 设备 | 设备名、OS、架构。 |
| 来源 | web / cli。 |
| IP | 最近登录 IP。 |
| 最近活跃 | 时间。 |
| 当前设备 | 当前设备显示绿色徽章。 |
| 操作 | 吊销。 |

交互：

- 当前设备不可单独吊销，只能通过退出登录。
- 吊销其他设备弹窗显示设备名、来源、最近活跃时间。
- 批量“注销其他设备”保留当前设备。

安全细节：

- 新设备登录通知从通知中心进入本页。
- 登录异常仍在审计日志页展示，本页只管理设备。

## 交互补充

- 登录失败后保留账号输入，清空密码并聚焦密码框。
- 多组织用户登录后进入组织选择；选择组织时列表项进入 loading，避免重复点击。
- 创建组织成功后自动切换到新组织，并进入项目列表。
- 邀请成员弹窗提交成功后保留弹窗关闭前的角色选择，便于连续邀请同角色成员。
- 吊销设备成功后该设备行从列表中移除；如果吊销的是其他设备，当前页面不跳转。
- “注销其他设备”成功后列表只保留当前设备，并显示成功 toast。
