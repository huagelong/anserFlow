```  | 字段 | 说明 | |------|------| | status | `success` 登录成功 / `failed` 登录失败 | | failReason | 失败原因：`invalid_password` / `user_not_found` / `account_locked` / `rate_limited` | | source | `web` 浏览器登录 / `cli` 命令行登录 | | deviceID | 与 `devices` 表关联，可追踪设备 |
