```  > **ER 图补充**：Comment、LoginLog、AuditLog、Device、IssueLabel、TestResult 为后续新增表，未在上方 ER 图中展示。完整表关系见各节 SQL DDL。  > **数据库方言**：以上 DDL 以 MySQL 语法展示。默认使用 SQLite（通过 GORM AutoMigrate 自动建表），无需手写 DDL。GORM 的 `gorm:"serializer:json"` tag 保证 JSON 字段在 SQLite (TEXT) 和 MySQL (JSON) 间透明切换。
