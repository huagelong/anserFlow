> 来源：`docs/01-项目概述.md` 第 3 行 > 位置：[总目录](../README.md) -> [01 - 项目概述](README.md) -> 当前章节 > 相邻：上一篇：无 | [下一篇](1.2-目标用户.md) > 相关主题：[02 - 系统架构](../02-系统架构/README.md) | [03 - 数据模型](../03-数据模型/README.md) | [04 - API 设计](../04-API设计/README.md) ## 1.1 flowcode 是什么  flowcode 是一套专为 **vibe coding** 工作流设计的 SaaS 需求管理系统。它填补了「灵感 → 需求 → 分析 → 计划 → AI编码 → PR → 上线 → 监控反馈」全链路的工具空白，让独立开发者和微型创意团队可以像聊天一样管理整个软件开发周期。  核心理念：**用对话驱动开发，用自动化消除繁琐**。  flowcode 同时提供 **Web 控制台**、**CLI 工具**、**RESTful API** 三种接入形态，并通过 **Skills 技能包** 扩展 AI 执行能力，满足从终端极客到 Web 用户的不同偏好。  flowcode 编译为单个纯静态跨平台二进制文件：Go 后端 + Vue 3 前端（`embed` 内嵌）+ SQLite，一个文件即完整系统。
