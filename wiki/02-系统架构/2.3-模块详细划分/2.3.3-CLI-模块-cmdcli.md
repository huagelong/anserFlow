CLI 入口 (Cobra) ├── cmd/ │   ├── root.go                    # 根命令 │   ├── server.go                  # flowcode server (启动 Web 服务) │   ├── login.go                   # flowcode login │   ├── init.go                    # flowcode init (一键初始化项目) │   ├── sync.go                    # flowcode sync (同步 config + skills 到 Git) │   ├── issue.go                   # flowcode issue [create|list|search|close|watch|...] │   ├── requirement.go             # flowcode req [create|list|convert] │   ├── test.go                    # flowcode test [run|result|list|watch] │   ├── workflow.go                # flowcode workflow [status|next|rollback] │   ├── history.go                 # flowcode history [list|diff|rollback] │   ├── pr.go                      # flowcode pr [create|list|status|merge] │   ├── skill.go                   # flowcode skill [install|list|create] │   ├── project.go                 # flowcode project [list] │   ├── stats.go                   # flowcode stats (项目健康面板) │   ├── audit.go                   # flowcode audit-log │   ├── upgrade.go                 # flowcode upgrade (CLI 自更新) │   └── config.go                  # flowcode config [set|get] └── pkg/     ├── client/                    # API 客户端封装     │   └── api.go     └── output/                    # 输出格式化 (JSON/Table/Color)         └── printer.go ```  **命令总览**：  | 命令组 | 子命令 | 说明 | 优先级 | |--------|--------|------|--------| | `login` | — | 登录认证，设备注册 | 🔴 核心 | | `server` | — | 启动 flowcode Web 服务（API + WebSocket） | 🔴 核心 | | `init` | — | 一键初始化：创建 `.flowcode/config.yaml` + sync skills + git commit | 🔴 高 | | `sync` | — | 将 skills + config.yaml 写入项目 Git 仓库 | 🔴 高 | | `issue` | `create`, `list`, `search`, `close`, `watch` | Issue 全生命周期管理 | 🔴 核心 | | `test` | `run`, `result`, `list`, `watch` | 测试执行与结果查询 | 🔴 高 | | `workflow` | `status`, `next`, `rollback` | 工作流状态查看与推进 | 🟡 中 | | `history` | `list`, `diff`, `rollback` | 历史记录查看 / 字段级 diff / 版本回滚 | 🟡 中 | | `req` | `create`, `list`, `convert` | 需求管理 | 🟡 中 | | `stats` | — | 项目健康面板：Issue 数 / 测试通过率 / 覆盖率 / 阻塞项 | 🟡 中 | | `pr` | `create`, `list`, `status`, `merge` | PR 创建、列表、状态追踪与合并 | 🟡 中 | | `skill` | `install`, `list`, `create` | Skill 管理 | 🟢 低 | | `project` | `list` | 项目列表 | 🟢 低 | | `config` | `set`, `get` | 本地配置管理 | 🟢 低 | | `audit-log` | — | 操作审计 + 登录日志 | 🟢 低 | | `upgrade` | — | CLI 工具自更新（自动检测 + 下载最新版本） | 🟢 低 |  **典型使用场景**：  ```bash # 1. 初始化项目（首次接入 flowcode） flowcode init # → ✓ Created .flowcode/config.yaml # → ✓ Synced 7 skills to .flowcode/skills/ # → ✓ Committed & pushed to git  # 2. 查看项目概览 flowcode stats # → Issues: 12 open / 5 in review / 3 blocked # → Tests: 78% pass rate / 62% coverage # → PRs: 2 open  # 3. 创建 Issue 并查看工作流状态 flowcode issue create -t "用户登录页" -d "实现邮箱+密码登录" -p p1 # → Created: iss_abc123 flowcode workflow status iss_abc123 # → State: draft | Next: submit / cancel  # 4. 运行测试 flowcode test run tc_xyz --watch # → [实] 实时流式输出测试日志...  # 5. 查看历史并回滚 flowcode history list --issue iss_abc123 flowcode history rollback h_old_001 --confirm  # 6. PR 管理 flowcode pr list                          # 项目所有 PR flowcode pr list --status open            # 只看打开的 flowcode pr create --issue iss_abc123 --branch feat/login flowcode pr merge --issue iss_abc123       # 合并 PR  # 7. 升级 CLI flowcode upgrade # → Current: v1.2.3 | Latest: v1.3.0 # → Downloading... Done → Restart to apply ```
