sha256=<HMAC(payload, secret)> X-FlowCode-Event: issue:approved ```  接收方验证（以 Node.js 为例）：  ```js const crypto = require('crypto'); const signature = req.headers['x-flowcode-signature']; const expected = 'sha256=' + crypto.createHmac('sha256', secret).update(JSON.stringify(req.body)).digest('hex'); if (crypto.timingSafeEqual(Buffer.from(signature), Buffer.from(expected))) {   // 验证通过 } ```  ####
