# 01 - 项目概述

## 1.1 flowcode 是什么

flowcode 是一套专为 **vibe coding** 工作流设计的个人开发工具。它填补了「想法 → Issue → AI编码 → PR → 上线」链路的工具空白，让你像聊天一样管理整个编码周期。

核心理念：**用对话驱动开发，用自动化消除繁琐**。

flowcode 同时提供 **Web 控制台** 和 **CLI 工具** 两种接入形态，并通过 **Skills 技能包** 扩展 AI 执行能力。

flowcode 编译为单个纯静态跨平台二进制文件：Go 后端 + Vue 3 前端（`embed` 内嵌）+ SQLite，一个文件即完整系统。

## 1.2 目标用户

| 用户角色 | 典型场景 |
|---------|---------|
| 独立开发者 | 一个人维护多个 side project，希望快速把想法变成代码并部署 |
| Indie Hacker | 验证 idea → 快速上线 → 收集反馈 → 持续迭代 |
| CLI 重度用户 | 终端内完成需求提交、查看分析/计划、触发执行、查看日志，不离开命令行 |

## 1.3 解决的核心痛点

1. **需求碎片化**：想法散落在聊天记录、备忘录中，缺乏统一入口
2. **编码执行断层**：提出 Issue 后仍需人工编码，AI 工具缺少调度中枢
3. **工具链割裂**：需求管理、AI 编码、Git 管理各自独立
4. **测试滞后**：AI 生成代码后缺乏自动测试验证，质量难以保证

## 1.5 核心竞争力

### 顺滑的 Issue 到代码体验
用自然语言描述需求创建一个 Issue，AI 自动完成分析、生成执行计划，直接在 Docker 沙盒中编码并提交 PR。

### 插件式 Skills 体系
Skills 以 Markdown 文件定义可复用的编码规范，AI 执行时自动加载。支持自定义 Skill，让 AI 按你的编码风格工作。

### 智能交互：人机协同决策
AI 执行编码遇到歧义（方案选择、缺失配置、边界确认）时，可主动暂停并向你提问。回答后 AI 继续执行，形成人机协同的闭环决策机制。

### 极简部署
默认 SQLite + DB 轮询队列，`./flowcode server` 即可启动完整服务，无需 MySQL、无需 Redis。一个文件即完整系统。

## 1.7 技术栈速览

| 层级 | 技术 |

| **前端** | Vue 3 + TypeScript + Vite + Element Plus + Pinia + Vue Router |

| **沙盒容器** | Docker SDK (安全执行 AI 生成代码) |

## 1.8 两种接入形态

### Web 控制台
基于 Vue 3 + Element Plus 的现代化管理界面，看板、实时日志流一应俱全。

### CLI 工具 (`flowcode`)

覆盖全生命周期的命令行工具，终端内完成所有操作。

**安装**：

```bash
# 下载二进制
curl -fsSL https://github.com/flowcode/flowcode/releases/latest/download/flowcode-linux-amd64 -o flowcode
chmod +x flowcode
```

**命令一览**：

| 命令组 | 核心子命令 | 场景 |
|--------|-----------|------|
| `issue` | create / list / watch / close | Issue 全生命周期 |
| `pr` | create / list / merge | PR 创建与合并 |
| `workflow` | status / rollback | 工作流推进与回滚 |
| `skill` | install / list / create | Skills 管理 |
| `init` / `sync` | - | 项目初始化与 Skills 同步 |
| `stats` / `history` | - | 概览 / 历史 |

```bash
# 首次接入
flowcode init

# 创建 Issue，系统先分析并生成计划，再进入执行
flowcode issue create "修复登录页面样式错乱" --priority p0

# 实时查看执行日志
flowcode issue watch abc123

# 项目健康概览
flowcode stats
# -> Issues: 12 open / 5 in review / 3 blocked
```

## 1.9 项目命名

**flowcode** = flow (流动/工作流) + code (编码)

寓意：灵感如流水般顺畅地转化为代码，享受 vibe coding 的心流体验。
